/*
===========================================================================================================
Description
===========================================================================================================
This project deals with using RS232 as input to control the Web Interface to control LED Lights. 
===========================================================================================================
*/

/*
===========================================================================================================
Usage
===========================================================================================================
Please make sure that you have connected the correct LED with 220ohm register and pin to 3 as output. When
uploaded, you will need to connect COM1 by using putty to read the information tht comes with this project.
This will then initialize and give you an IP address that you can use to load the web page. 
===========================================================================================================
*/
